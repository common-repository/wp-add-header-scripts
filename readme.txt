== Description ==
WP Add Header Scripts will insert a script that was set between HEAD tag and HEAD tag.
[日本語の説明を読む](http://www.puniokid.com/)


== Installation ==
1. Upload the `wp-add-header-scripts` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Please set, script that you want to view, order, and an upper limit on the number.
